# Skill 上线检查清单

## 基本检查
- [ ] 触发场景清晰
- [ ] 输入要求清晰
- [ ] 输出要求清晰
- [ ] 边界写清楚
- [ ] 不适用场景写清楚

## 可执行性检查
- [ ] 至少有 3 个真实样本支持
- [ ] 步骤具有重复性
- [ ] 有明确节省价值
- [ ] 失败时可人工回退
- [ ] 不依赖脆弱或不稳定前置条件

## 维护性检查
- [ ] 维护成本可接受
- [ ] 不强绑定临时上下文
- [ ] 不依赖复杂、不可控外部条件
- [ ] 后续迭代路径明确

## 上线前结论
- [ ] 建议上线
- [ ] 建议继续观察
- [ ] 建议暂缓
